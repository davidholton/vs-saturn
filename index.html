Saturn is a Visual Studio Code extension that promotes time management and productivity through the Pomodoro technique.

Install

Background

Time is a challenge for many of us to manage. Programmers are racing against the clock to finish assignments and meet deadlines. Without a clear plan it becomes an endeavor for those to manage it themselves. The Pomodoro Technique was created to let you work with time, instead of struggle against it.

Our goal was to bring the Pomodoro Technique to programmers and help promote time management to them. By implementing the system into an extension for Visual Studio Code the user has the benefit of easily using the system whenever they work.

Description

Built with Visual Studio Code Extension API VS Saturn will provide all the necessary components of the Pomodoro technique in a nonintrusive and streamlined design.

Featuring a small timer for the current work cycle with a pause and snooze button if the user needs to pause or delay working. Tally marks so the user can visually see their current progress on their work cycle. A reset button so the user may restart to the beginning of a work cycle. VS Saturn also includes a nonintrusive notification of when a cycle ends by color theme change or notification. And additionally a separate interactive task list for both their work and break cycles to keep track of what needs to get done.
