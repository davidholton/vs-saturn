Saturn is a Visual Studio Code extension that promotes time management and productivity through the Pomodoro technique.

Install

About

Saturn will provide all the necessary components for pomodoro in a nonintrusive and streamlined design. This includes: A small timer for the current cycle with a pause/snooze feature for the user if they need to step away for a while. A tally mark system so the user can visually see which current cycle they are on. Reset functionality so the user can reset entirely from their very first work cycle. Separate, interactive task lists for the work and break cycles so the user can keep track of what needs to be done. A Nonintrusive notification of when a cycle ends by either a tone or changing the border color of VSCode.
