How to install and run the prototype builds

Prerequisites

Firstly Visual Studio Code is required for running the extension on. Links to each platform can be found on the Visual Studio website.

  1. https://code.visualstudio.com/download

Installer executables and Linux binaries for Node.js can be found on their website download page. No special install options are required.

  1. https://nodejs.org/en/download

Then download and unpack the latest Prototype release from the GitHub repository.

    Prototype v1

  1. https://github.com/davidholton/vs-saturn/releases/tag/prototype-v1.0

    2. Prototype v2

  2. Unreleased

Installation and Running

Once unpacked run npm install inside the unpacked prototype folder. Open the prototype project in Visual Studio Code by either dragging the unpacked prototype folder onto an open Visual Studio Code application or running code . in a terminal inside the unpacked prototype folder if Visual Studio Code is recognized by your PATH.


To run the extension hit “F5” while in Visual Studio Code or from the menu bar click Run > Start Debugging. From there a Visual Studio Code Extension Development Host window will appear. In the bottom left Saturn’s interface will appear and commands will be found in the command palette (Ctrl+Shift+P or Cmd+Shift+P on OS X). Press the play button to start Saturn.
